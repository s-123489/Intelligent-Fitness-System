# 运行说明

## Windows用户运行步骤

### 1. 安装JDK 17

下载并安装JDK 17: https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html

验证安装:
```cmd
java -version
```

应该显示: java version "17.x.x"

### 2. 安装Maven (可选，使用Maven Wrapper则不需要)

本项目包含Maven Wrapper，无需单独安装Maven。

### 3. 运行后端

在`backend-java`目录下执行:

**Windows CMD:**
```cmd
mvnw.cmd clean spring-boot:run
```

**Windows PowerShell:**
```powershell
.\mvnw.cmd clean spring-boot:run
```

### 4. 验证启动成功

看到以下输出表示启动成功:
```
========================================
智能健身管理系统启动成功！
API地址: http://localhost:5000/api
H2控制台: http://localhost:5000/api/h2-console
========================================
```

### 5. 测试API

使用Postman或浏览器测试:

注册用户:
```bash
curl -X POST http://localhost:5000/api/register \
  -H "Content-Type: application/json" \
  -d "{\"username\":\"testuser\",\"password\":\"123456\"}"
```

登录:
```bash
curl -X POST http://localhost:5000/api/login \
  -H "Content-Type: application/json" \
  -d "{\"username\":\"testuser\",\"password\":\"123456\"}"
```

## 常见启动问题

### 端口被占用
修改 `src/main/resources/application.yml`:
```yaml
server:
  port: 8080  # 改为其他端口
```

### 内存不足
设置JVM参数:
```cmd
set MAVEN_OPTS=-Xmx512m -Xms256m
mvnw.cmd spring-boot:run
```

### 编译错误
清理并重新编译:
```cmd
mvnw.cmd clean install -DskipTests
mvnw.cmd spring-boot:run
```
